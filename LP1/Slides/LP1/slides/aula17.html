Biblioteca Boost

  • Coleção de bibliotecas multiplataforma de código aberto em C++
  • Criada em 1998 por diversos membros do comitê de padronização de C++, mas atualmente é mantida por mais de 3000 colaboradores de todas as partes do mundo
  • Diversas bibliotecas de Boost têm sido aceitas para serem incorporadas na biblioteca padrão de C++
  • Contém mais de 100 módulos organizados em mais ou menos 20 categorias, sendo 80% deles definidos em cabeçalhos
  • Para garantir eficiência e flexibilidade, ela faz uso extensivo de templates
  • Versão atual: 1.55.0 (http://www.boost.org)

  • Algoritmos
  • Auxiliares para compiladores que não seguem o padrão

  • Estrutura de dados
  • Objetos-função e programação de alta-ordem
  • Programação genérica
  • Grafos
  • Entrada/saída de dados
  • Suporte inter-linguagem
  • Iteradores
  • Matemática e sistemas numéricos
  • Uso e gerenciamento de memória
  • Análise sintática
  • Meta-programação de pré-processadores

  • Ponteiros inteligentes
  • Processamento de cadeias de caracteres

Módulos da biblioteca Boost

Exemplo de uso de Boost (1)

  • boost::noncopyable
    • Indica que uma classe não deve ser copiável (ambos via o construtor de cópia e o operador de atribuição)





    • Proibe operações de cópia somente nos casos em que os operadores de cópia não forem definidos pelo usuário na classe derivada
#include <boost/noncopyable.hpp>

// ...

class MyClass : private boost::noncopyable
{
    /*...*/
};

Exemplo de uso de Boost (1)

  • boost::noncopyable
namespace boost
{
    class noncopyable
    {
    protected:
        noncopyable() {}
       ~noncopyable() {}

    private: // enfatiza que os membros são privados!
        noncopyable( const noncopyable& );
        const noncopyable& operator=( const noncopyable& );
    };
}

Exemplo de uso de Boost (2)

  • boost::shared_ptr (ponteiro compartilhado)
    • Quando existem múltiplos clientes para um objeto, mas nenhum é o proprietário explícito
    • Quando precisamos armazenar ponteiros em contêineres
    • Quando passamos ou recebemos objetos de bibliotecas sem conhecermos o proprietário dos mesmos
#include <boost/shared_ptr.hpp>

int main()
{
    boost::shared_ptr< int > ptr( new int( 42 ) );
    *ptr = 36;

    return 0;
}

Exemplo de uso de Boost (3)

  • BOOST_STATIC_ASSERT
    • Restrições em tempo de compilação com expressões constantes
    • Encerra o processo de compilação quando a condição não é válida
    • Não cria tempo de latência na execução caso a condição for válida
#include <boost/static_assert.hpp>

int main()
{
    const int config = 0;
    typedef int real;

    // Condição válida em tempo de compilação
    BOOST_STATIC_ASSERT( config == 0 );
    // Condição inválida, o processo sendo imediatamente encerrado!
    BOOST_STATIC_ASSERT( boost::is_floating_point< real >::value );

    return 0;
}

Exemplo de uso de Boost (4)

  • boost::numeric::ublas
    • Módulo de álgebra linear que dá suporte ao uso de vetores e matrizes
    • Capaz de fazer interface com diversas bibliotecas numéricas como ATLAS, BLAS e LAPACK
#include <boost/numeric/ublas/vector.hpp>
#include <boost/numeric/ublas/matrix.hpp>
#include <boost/numeric/ublas/io.hpp>
#include <iostream>
 
int main () // Exemplo do "y = Ax"
{
    boost::numeric::ublas::vector< double > x( 2 );
    x( 0 ) = 1; x( 1 ) = 2;
    boost::numeric::ublas::matrix< double > A( 2, 2 );
    A( 0, 0 ) = 0; A( 0, 1 ) = 1;
    A( 1, 0 ) = 2; A( 1, 1 ) = 3;
    boost::numeric::ublas::vector< double > y = boost::numeric::ublas::prod( A, x );
    std::cout << y << std::endl;
    return 0;
}

Exemplo de uso de Boost (5)

  • boost::thread
    • Aplicações que necessitam executar procedimentos independentes em paralelo (multi-threading)
#include <boost/thread/thread.hpp>
#include <iostream>
 
void imprimir() {
    std::cout << "Ola, eu sou um thread!" << std::endl;
}

int main()
{
    // inicia 2 threads que chamam a sub-rotina "imprimir"
    boost::thread thread1( &imprimir );
    boost::thread thread2( &imprimir);
    // espera que os dois threads terminem a execução
    thread1.join();
    thread2.join();
    return 0;
}

Biblioteca Qt

  • Biblioteca de classes para o desenvolvimento de interfaces gráficas de usuário multiplataforma em C++
  • 1991: início do desenvolvimento
    • Necessidade de conjunto de ferramentas para GUI multiplataforma
  • 1994: Início da comercialização pela Trolltech
    • "Q" de boa aparência em fonte Emacs e "t" de toolkit
    • Possui licenças comerciais e de código aberto
  • 2008: Adquirida pela Nokia quando passou a se chamar Qt Development Frameworks
    • Foco na plataformas móveis
  • 2012: Adquirida pela Digia
    • Versão atual: 5.1 ( http://qt.digia.com e http://qt-project.org)

Termos de Qt

  • Widgets
    • Componentes de GUI (botões, caixas de mensagem, janelas de aplicação)
  • Layout Manager
    • Posicionamento e redimensionamento automático dos widgets dependentes
  • Signals e Slots
    • Comunicação entre objetos
  • Events
    • Eventos do sistema (cliques do mouse, teclado, ...)
  • Actions
    • Ações disparadas pelas barras de ferramentas e menus

Exemplo de uso de Qt

  • Aplicação para criar uma janela gráfica contendo no seu interior um texto e uma caixa de edição
#include <QtApplication>
#include <QLabel>
#include <QTextEdit>
 
int main( int argc, char *argv[] )
{
    QApplication app( argc, argv );
    QLabel label( "Estou usando Qt!" );
    QTextEdit textEdit;

    label.show();
    textEdit.show();
    return app.exec();
}
