Recapitulativo (2)

  • Estrutura padrão de código fonte de um programa em linguagem C
#include <"arquivo cabeçalho">

int main( void )
{

  • Cada bloco de dados contém:
    • um endereço físico na memória
    • uma etiqueta para indicar o significado do seu conteúdo

0xFFAD854C

0xF49B112F

  • Comandos de entrada e saída

Recapitulativo (7)

  • Sub-rotinas: forma de melhor estruturar o código
    • Evitam a sobrecarga do programa por eliminar uma única sequência de instruções dentro do método main()
    • Permitem reaproveitar código já desenvolvido anteriormente
    • Facilitam o entendimento pelo fato de criarem códigos mais curtos
  • Uma sub-rotina pode fazer chamadas a outras sub-rotinas, assim como a ela própria, durante a sua execução
  • A implementação pode ser iterativa ou recursiva

Recapitulativo (8): boas práticas de programação

  • Manter uma aparência mais legível possível ao código
    • Facilitar o desenvolvimento e a manutenção dos programas
  • Recomenda-se
    • Escrever um comando por linha
    • Alinhar os comandos dentro de um bloco
#include <stdio.h>
int main( void )
{
    printf( "O jogo iniciou!\n" );
    do {
        /* Cada iteração do laço executa um conjunto de instruções */
    } while( 1 );
    printf( "O jogo terminou!\n" );
    return 0;
}

Linguagem C++

Linguagem C++

  • C++ é uma linguagem de programação multi-paradigma
    de propósito geral padronizada pela ISO
  • Ela é considerada de médio nível, pois combina
    características de linguagens de alto e baixo níveis
  • C++ foi criada por Bjarne Stroustrup no AT&T Bell Labs
    no início dos anos 80
  • Após a padronização ISO de 1998 e a posterior revisão
    de 2003, uma nova versão da especificação da linguagem,
    conhecida como C++11, foi lançada em 2011

Bjarne Stroustrup

Linguagem C++

  • Principais objetivos
    • Inserir o paradigma orientado a objetos em C
    • Manter-se simultaneamente próxima da máquina
      e (da análise) do problema
  • Por quê?
    • A medida que os sistemas de software crescem, também cresce a complexidade associada a eles
      • Torna-se difícil satisfazer um grande número de requisitos
    • O paradigma orientado a objetos oferece uma nova forma para tratar essa complexidade
      • Organiza o código em componentes lógicos que facilitam a programação

Paradigma Orientado a Objetos

  • Paradigma de programação que permite
    aos programadores racionar e solucionar
    problemas em termos de objetos
    diretamente associados às entidades reais

  • A programação orientada a objetos serve
    de elo entre os problemas existentes e as
    soluções computacionais
    • Grande importância na solução de problemas complexos

Comparativo entre C, C++ e Java

  • C, C++ e Java são as linguagens de programação
    comerciais mais populares (TIOBE)
    • C segue o paradigma procedural
      • Soluções baseadas na decomposição de tarefas distintas
    • Java segue o paradigma orientado a objetos
      • Soluções baseadas na decomposição de objetos distintos
    • C++ é híbrida (ou multi-paradigma), permitindo seguir os paradigmas procedural e orientado a objetos
      • Partes da solução baseadas em tarefas distintas e outras partes em objetos distintos

Comparativo entre C e C++ (1)

  • C++ é uma extensão da linguagem C
    • Contém um superconjunto de C, no qual quase toda instrução correta em C é correta em C++


    • Tem todas as vantagens do C, além de permitir abstração de dados e manipulação de objetos

  • Ela também é bastante usada na academia devido ao seu excelente desempenho e uma grande base de usuários

Componentes da linguagem C++ (1)

  • A inclusão de cabeçalhos com a diretiva #include diz ao compilador para inserir um outro arquivo no código fonte
    • Em C++, ela não necessita mais da extensão do arquivo (.h)
    • Na biblioteca padrão de C++, iostream substitui stdio.h de C
  • Comentários iniciam com // e terminam no fim da linha

Exemplo de código em linguagem C

Exemplo de código em linguagem C++

#include <stdio.h>

int main( void )
{
    /* Comentário no estilo
       de C */
    return 0;
}

#include <iostream>

int main()
{
    // Comentário no estilo de C++
    /* Comentário no estilo de C
       também é aceito em C++ */
    return 0;
}

Componentes da linguagem C++ (2)

  • Comando de fluxo de saída padrão
    • std::cout substitui printf, eliminando os identificadores %
    • << é um operador de inserção que direciona o valor a ser impresso para o dispositivo de saída
    • O manipulador std::endl substitui o caractere '\n'

Exemplo de código em linguagem C

Exemplo de código em linguagem C++

#include <stdio.h>

int main( void )
{
    int x = 10;
    printf( "Iniciando...\n" );
    printf( "%i, %f\n", x, 20.5f );
    return 0;
}

#include <iostream>

int main()
{
    int x = 10;
    std::cout << "Iniciando..." << std::endl;
    std::cout << x << ", " << 20.5f << std::endl;
    return 0;
}

Componentes da linguagem C++ (3)

  • Comando de fluxo de entrada padrão
    • std::cin substitui scanf, no qual identificadores % e operador de endereçamento & não são mais necessários
    • >> é um operador de extração que recebe um valor digitado pelo usuário através do dispositivo de entrada

Exemplo de código em linguagem C

Exemplo de código em linguagem C++

#include <stdio.h>

int main( void )
{
    int x;
    float y;
    scanf( "%i %f", &x, &y );
    return 0;
}

#include <iostream>

int main()
{
    int x;
    float y;
    std::cin >> x >> y;
    return 0;
}
