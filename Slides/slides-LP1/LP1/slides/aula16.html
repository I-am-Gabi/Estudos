Recapitulativo: tipos de contêineres

  • Contêineres sequenciais
    • Armazenam elementos em um arranjo estritamente linear
    • Exemplos: vector, deque, list
  • Contêineres associativos
    • Armazenam elementos como chaves ordenadas que podem ser rapidamente recuperadas
    • Exemplos: set, multiset, map, multimap
  • Contêineres adaptadores
    • Armazenam elementos através de outros contêineres com suas estruturas de armazenamento subjacentes
    • Exemplos: stack, queue, priority_queue

Algoritmos da STL

  • Algoritmos da STL são procedimentos que permitem processar dados dos contêineres
  • Eles existem em diversos tipos (cabeçalho <algorithm>):
    • Não modificadores: não mudam nem a ordem nem o valor dos elementos
    • Modificadores: mudam o valor dos elementos
    • Removedores: removem elementos
    • Mutadores: mudam a ordem dos elementos
    • Ordenadores: ordenam elementos usando um critério
    • Numéricos: combinam elementos numéricos de diferentes maneiras

Exemplos de algoritmos

Nome

 Efeito
for_each() Executa uma operação para cada elemento de um contêiner
find() Busca o primeiro elemento de um contêiner segundo um valor
search() Busca a primeira ocorrência de um intervalo
copy() Copia elementos de um intervalo em um contêiner
fill() Substitui elementos de um contêiner com um valor dado
replace() Substitui elementos específicos de um contêiner com um valor dado
remove() Remove elementos segundo um valor dado
reverse() Inverte a ordem dos elementos de um contêiner
sort() Ordena todos os elementos de um contêiner
merge() Une os elementos de dois intervalos
accumulate() Combina todos os elementos de um contêiner

Algoritmo for_each()

  • Executa uma operação para cada elemento de um contêiner
    • Sintaxe: for_each( Iterator begin, Iterator end, UnaryProc op )
#include <vector>
#include <algorithm>
#include <iostream>

void imprimir( int n )
{
    std::cout << n << " ";
}

int main()
{
    int arranjo[] = { 12, 3, 17, 8, 34, 56, 9 };
    std::vector< int > vetor( arranjo, arranjo + 7 );
    
    // aplica a função mostrar para cada elemento do vetor
    std::for_each( vetor.begin(), vetor.end(), imprimir );

    return 0;
}

Algoritmo find()

  • Busca o primeiro elemento de um contêiner segundo um valor
    • Sintaxe: find( Iterator begin, Iterator end, const T& value )
#include <vector>
#include <algorithm>
#include <iostream>

int main() {
    int valor;
    int arranjo[] = { 12, 3, 17, 8, 34, 56, 9 };
    std::vector< int > vetor( arranjo, arranjo + 7 );
    std::vector< int >::iterator iter;

    std::cout << "Digite um valor: ";
    std::cin >> valor;
    iter = std::find( vetor.begin(), vetor.end(), valor ); // busca valor em vetor

    if( iter != vetor.end() ) // se elemento encontrado
        std::cout << "Elemento " << valor << " foi encontrado!" << std::endl;
    else
        std::cout << "Elemento " << valor << " nao foi encontrado!" << std::endl;
    return 0;
}

Algoritmo find_if()

  • Busca o primeiro elemento de um contêiner segundo um critério
    • Sintaxe: find_if( Iterator begin, Iterator end, UnaryPredicate op )
#include <vector>
#include <algorithm>
#include <iostream>

bool teste( int n ) {
    return ( n > 21 ) && ( n < 36 );
}
int main() {
    int arranjo[] = { 12, 3, 17, 8, 34, 56, 9 };
    std::vector< int > vetor( arranjo, arranjo + 7 );
    std::vector< int >::iterator iter;

    // busca elemento em vetor quando o predicado teste for verdadeiro
    iter = std::find_if( vetor.begin(), vetor.end(), teste );
    if( iter != vetor.end() ) // se elemento encontrado
        std::cout << "Elemento " << *iter << " foi encontrado!" << std::endl;
    else
        std::cout << "Nenhum elemento foi encontrado!" << std::endl;
    return 0;
}

Algoritmo count_if()

  • Calcula número de elementos num contêiner segundo um critério
    • Sintaxe: count_if( Iterator begin, Iterator end, UnaryPredicate op )
#include <vector>
#include <algorithm>
#include <iostream>

bool teste( int n ) {
    return ( n > 14 ) && ( n < 36 );
}

int main()
{
    int arranjo[] = { 12, 3, 17, 8, 34, 56, 9 };
    std::vector< int > vetor( arranjo, arranjo + 7 );

    // conta elementos em vetor quando teste for verdadeiro
    int n = std::count_if( vetor.begin(), vetor.end(), teste );

    std::cout << "Encontrado " << n << " elemento(s)!" << std::endl;
    return 0;
}

Algoritmos sort() e merge()

  • sort: permite ordenar elementos de um contêiner
  • merge: permite unir elementos de dois contêineres








  • Os algoritmos podem estar inclusos ou não nos contêineres
#include <list>
#include <algorithm>

int main()
{
    int arranjo1[] = { 6, 4, 9, 1, 7 };
    int arranjo2[] = { 4, 2, 1, 3, 8 };
    std::list< int > lista1( arranjo1, arranjo1 + 5 );
    std::list< int > lista2( arranjo2, arranjo2 + 5 );

    lista1.sort();           // lista1 = { 1, 4, 6, 7, 9 }
    lista2.sort();           // lista2 = { 1, 2, 3, 4, 8 }
    lista1.merge( lista2 );  // lista1 = { 1, 1, 2, 3, 4, 4, 6, 7, 8, 9 }
                             // lista2 = {}
    return 0;
}

Algoritmo copy()

  • Copia elementos de um intervalo em um contêiner
    • Sintaxe: copy( Iterator srcBegin, Iterator srcEnd, Iterator destBegin )
#include <list>
#include <algorithm>

int main()
{
    int arranjo1[] = { 1, 3, 5, 7, 9 };
    int arranjo2[] = { 2, 4, 6, 8, 10 };
    std::list< int > lista1( arranjo1, arranjo1 + 5 );
    std::list< int > lista2( arranjo2, arranjo2 + 5 );

    // copia o conteúdo de lista1 em lista2, sobrescrevendo os elementos existentes
    std::copy( lista1.begin(), lista1.end(), lista2.begin() );

    return 0;
}

Algoritmo copy()

  • Podemos também usar iteradores predefinidos
    • back_inserter: insere novos elementos no fim
    • front_inserter: insere novos elementos no início (elemento por elemento)
    • inserter: insere novos elementos em uma posição específica (todos de uma vez)
#include <list>
#include <algorithm>

int main() {
    int arranjo1[] = { 1, 3, 5, 7, 9 };
    int arranjo2[] = { 2, 4, 6, 8, 10 };
    std::list< int > lista1( arranjo1, arranjo1 + 5 );
    std::list< int > lista2( arranjo2, arranjo2 + 5 );

    // adiciona o conteúdo de lista1 no final de lista2
    std::copy( lista1.begin(), lista1.end(), std::back_inserter( lista2 ) );
    // adiciona o conteúdo de lista1 no início de lista2
    std::copy( lista1.begin(), lista1.end(), std::front_inserter( lista2 ) );
    // adiciona o conteúdo de lista1 no início de lista2
    std::copy( lista1.begin(), lista1.end(), std::inserter( lista2, lista2.begin() ) );
    return 0;
}

Objetos-função da STL

  • Objetos-função são objetos de uma classe template que contém um único método membro
    • operator()
  • Eles servem para definir critérios de processamento para certos algoritmos da STL
    • Alguns exemplos são sort, merge e accumulate que podem ter objetos-função como argumento
  • Na STL, existem diversos objetos-função predefinidos (cabeçalho <funcional>), mas o usuário também podendo criar os seus próprios

Exemplos de objetos-função predefinidos

Expressão

 Efeito
negate< type >() - parâmetro
plus< type >() parâmetro1 + parâmetro2
minus< type >() parâmetro1 - parâmetro2
equal_to< type >() parâmetro1 == parâmetro2
not_equal_to< type >() parâmetro1 != parâmetro2
less< type >() parâmetro1 < parâmetro2
greater< type >() parâmetro1 > parâmetro2
less_equal< type >() parâmetro1 <= parâmetro2
greater_equal< type >() parâmetro1 >= parâmetro2
logical_and< type >() parâmetro1 && parâmetro2
logical_or< type >() parâmetro1 || parâmetro2

Usando objetos-função (1)

#include <list>
#include <functional>
#include <numeric>
#include <iostream>

int main()
{
    int arranjo[] = { 6, 4, 9, 1, 7 };
    std::list< int >  lista( arranjo, arranjo + 5 ); // inicializa lista com arranjo

    // usa o objeto-função greater< int > para ordenar em ordem reversa
    lista.sort( std::greater< int >() );             // lista = { 9, 7, 6, 4, 1 }

    // soma por default
    int soma1 = accumulate( lista.begin(), lista.end(), 0 );
    // soma com o objeto-função std::plus
    int soma2 = accumulate( lista.begin(), lista.end(), 0, std::plus< int >() );
    // multiplicação com o objeto-função std::multiplies
    int mult  = accumulate( lista.begin(), lista.end(), 1, std::multiplies< int >() );

    std::cout << soma1 << std::endl << soma2 << std::endl << mult << std::endl;
    return 0;
}

Usando objetos-função (2)

#include <list>
#include <functional>
#include <numeric>
#include <iostream>

template< class T >
class squared_sum  // objeto-função definido pelo usuário
{
public:
    T operator()( T n1, T n2 ) { return n1 + n2 * n2; }
}; 

int main()
{
    int arranjo[] = { 6, 4, 9, 1, 7 };
    std::list< int >  lista( arranjo, arranjo + 5 ); // inicializa lista com arranjo

    // calcula a soma dos quadrados de uma lista de números inteiros
    int soma = accumulate( lista.begin(), lista.end(), 0, squared_sum< int >() );
    std::cout << soma << std::endl;

    return 0;
}

Contêineres associativos

  • Nos contêineres associativos, os elementos não são organizados em sequência, mas através de árvores ou tabelas hash
  • A vantagem principal destes contêineres é a rapidez da busca
  • A busca é feita usando uma chave que é um atributo do contêiner

Contêiner set< T >

#include <string>
#include <set>
#include <iostream>

int main()
{
    std::string names[] = { "Francisco", "Luis", "Joao", "Mario", "Guido" };
    std::set< std::string, std::less< std::string > > nameSet( names, names + 5 );

    // cria um conjunto de nomes com elementos ordenados alfabeticamente
    nameSet.insert( "Patricia" ); // inserção de mais nomes
    nameSet.insert( "Maria" );
    nameSet.erase( "Joao" );      // remove um elemento

    std::string searchName;
    std::cin >> searchName;

    std::set< std::string, std::less< std::string > >::iterator iter;
    iter = nameSet.find( searchName );
    if( iter == nameSet.end() )
        std::cout << searchName << " nao encontrado!" << std::endl;
    else
        std::cout << searchName << " encontrado!" << std::endl;
    return 0;
}

Contêiner map< T1, T2 >

#include <string>
#include <map>
#include <iostream>

int main()
{
    std::string names[] = { "Francisco", "Luis", "Joao", "Mario",
                            "Guido", "Patricia", "Maria", "Anne" };
    int numbers[] = { 75643, 83268, 97353, 87353, 19988, 76455, 77443, 12221 };

    std::map< std::string, int, std::less< std::string > > phonebook;
    std::map< std::string, int, std::less< std::string > >::iterator iter;

    for( int j = 0; j < 8; j++ )
        phonebook[names[j]] = numbers[j];

    for( iter = phonebook.begin(); iter != phonebook.end(); iter++ )
        std::cout << iter->first << " : " << iter->second << std::endl;

    std::cout << "Numero de Mario: " << phonebook["Mario"] << std::endl;

    return 0;
}
